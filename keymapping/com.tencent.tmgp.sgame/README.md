# 金铲铲之战 键盘映射

## 键位说明

| 按键 | 功能 |
|------|------|
| 空格 | 刷新商店 (D 牌) |
| F | 购买经验 (升级) |
| 1-5 | 购买对应位置的英雄 |
| Z | 锁定/解锁商店 |
| R | 快捷键物品栏 |
| ESC | 退出/取消 |

## 使用说明

1. 下载 `.playmap` 文件
2. 在 PlayCover 中右键点击金铲铲之战图标
3. 选择 "设置" → "键盘映射"
4. 点击左下角的导入按钮，选择下载的 `.playmap` 文件
5. 应用并保存

## 注意事项

- 此键位映射适用于 16:9 分辨率
- 如需调整键位位置，可在 PlayCover 的键盘映射编辑器中拖动按钮
- 出售英雄需要手动拖动英雄到下方出售区
- 查看其他玩家需要点击右侧玩家头像
